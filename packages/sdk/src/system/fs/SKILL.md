# Filesystem

File system operations for reading, writing, and managing files within sandboxes.

## Capabilities

- Read files and directories
- Write and create files
- List directory contents
- File metadata operations

## When to Use

- User needs to read or write files
- Directory operations required
- File system exploration
- Code file manipulation
