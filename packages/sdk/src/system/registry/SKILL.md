# Registry

Manages agent and skill registration, discovery, and lifecycle.

## Capabilities

- Register agents and skills
- Discover available agents
- Query skill capabilities
- Agent metadata management

## When to Use

- Need to find available agents
- Registering new capabilities
- Querying agent information
