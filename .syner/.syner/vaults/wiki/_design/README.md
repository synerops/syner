# Design Thinking

Reflexiones personales sobre diseño. Ideas en proceso.

## Propósito

Espacio para pensar sobre diseño sin presión de que sea "útil" inmediatamente. Aquí van:

- Observaciones vagas que podrían ser algo
- Conexiones entre conceptos que estás explorando
- Preguntas sin respuesta todavía
- Inspiración de otros proyectos/sistemas

## Diferencia con `design/vaults/_learnings/`

| Este vault | _learnings |
|------------|------------|
| Para pensar | Para actuar |
| Raw, incompleto | Destilado, operacional |
| Lo lees tú | Lo lee el agente |
| Puede contradecirse | Debe ser consistente |

## Flujo

```
Idea vaga aquí
    ↓ la revisitas varias veces
    ↓ empieza a tener forma
/syner-grow-note
    ↓ se vuelve documento real
    ↓ si es operacional →
design/vaults/_learnings/
```

## Convención

No hay estructura forzada. Organiza como te funcione. El filesystem es tu índice.
