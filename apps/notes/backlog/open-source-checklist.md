# open-source projects checklist

Crítico

-[] `LICENSE` - Sin licencia = nadie puede usar tu código legalmente
-[] `CONTRIBUTING.md` - Cómo contribuir al proyecto

Importante

-[] `CODE_OF_CONDUCT.md` - Normas de comportamiento
-[] `SECURITY.md` - Cómo reportar vulnerabilidades
-[] `.github/ISSUE_TEMPLATE/` - Templates para issues
-[] `.github/PULL_REQUEST_TEMPLATE.md` - Template para PRs

Nice to have

-[] Badges en `README` (licencia, CI status, etc.)
-[] Sección "Getting Started" más detallada
-[] Link al proyecto desplegado (si aplica)
